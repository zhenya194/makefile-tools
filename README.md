# makefile-tools

This is an utility that can help you delete not essential directories and files.

## Usage

### Makefile

0. Install `make`(if hadn't done it yet)
1. In your terminal and directory with `Makefile` file write `make NAME_OF_COMMAND`

### Python script

0. Install `python`(if hadn't done it yet)
1. In your terminal and directory with `script.py` file write `python script.py`
